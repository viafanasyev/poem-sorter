Detailed Description

Represents a text file mapped by mmap function. mmap is called during a constructor invocation. munmap is called in destructor. If the mapping fails, textPtr is set to nullptr and textSize is set to 0. Otherwise, textPtr points to a text start and textSize is set to a text size in bytes.

Maps the given file using mmap function. If the mapping fails, textPtr is set to nullptr and textSize is set to 0. Constructor also ensures that the given file is POSIX-like (ends with '\n') - just adds '\n' to the end of the text.
